Ribbons

=========================================================

Description:

This is a screen saver from Windows 8.1. This is a safe file.

=========================================================

Instructions:

Method 1:

1. Run AnomalyDetection.exe
2. Select Ribbons.scr
3. Click "Create file to scan"
4. Run Ribbons_injected.scr
5. Click "View Report".

Method 2:

1. Run AnomalyDetection.exe
2. Go to "Results" directory
3. Run "Ribbons_injected.scr"
4. Click "View Report".
